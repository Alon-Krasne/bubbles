# Dev Visual Controls

Use these while tuning animations/background.

## URL params
- `?dev=1` — enable dev overlay
- `?quality=high|medium|low` — initial quality tier
- `?bgClouds=0` — disable cloud system
- `?bgGrass=0` — disable grass sway system
- `?bgParticles=0` — disable ambient/weather particles

## Keyboard (dev mode)
- `Q` — cycle quality tier (`high -> medium -> low -> high`)
- `E` — trigger a strong manual background burst (for VFX tuning)
- Any key during intro — skip intro sequence

## Persisted localStorage keys
- `bubble_dev_mode`
- `bubble_quality`
- `bubble_bg_clouds`
- `bubble_bg_grass`
- `bubble_bg_particles`
